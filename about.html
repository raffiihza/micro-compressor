What is PentaServ?

PentaServ is a small web application to help users with several online tools that can be accessed for free.

PentaServ was released in April 2024. It was developed by a team of four people from SI-45-08 class (mentioned below) as part of their mid-term project for Enterprise Application Integration class. The team utilized a variety of technologies to build PentaServ. For the frontend, they used HTML, CSS, and Javascript, enhanced with the Bootstrap framework. The backend was developed using Python and Javascript, and Docker was employed to containerize the backend instances. The hosting of the application was done on GitHub Pages for the frontend, while Render was used for backend instances with Python, and Cloudflare Workers for backend instances with Javascript. This diverse set of technologies and platforms ensured a robust and efficient application.
